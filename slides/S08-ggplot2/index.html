What is ggplot2?

ggplot2 is a data visualization package for the statistical programming language R.

Basic Principles of ggplot2

To create, we first need to specify the data in the ggplot() function.
Then specify the aesthetic elements.
Then the type of plot.

Example:

# Load package
library(ggplot2)

# Upload Data
data <- read.csv("D:/R/SAR/data/pulse_data.csv", stringsAsFactors = T)

# Basic Code
#
#  ggplot(data)+
#    aes(x = var_x, y = var_y)+
#    geom_plot()

Standard of Coding

What is the standard of code writing?

Bar Chrat

Simple Bar Charts

  • A barplot shows the relationship between a numeric and a categorical variable.
  • Each entity of the categoric variable is represented as a bar.
  • The size of the bar represents its numeric value.

Purpose Simple Bar Charts

  • To describe the number of observations in each category of the discrete variable.
  • To visualize estimated error for discrete variables.

Simple Bar Chart (Vertical Orientation)

# For a Single Categorical Variable
library(ggplot2)
data |> 
  ggplot(aes(x = BMICat))+
  geom_bar()

Simple Bar Charts (Horizontal)

data |> 
  ggplot(aes(x = BMICat))+
  geom_bar()+
  coord_flip()

Sorting Bar Chart

  • For sort down ascending order we can just use fct_reorder() from forcats package.
  • .fun = length counts the number of occurrences for each level in BMICat.
data |> 
  ggplot(aes(x = forcats::fct_reorder(BMICat, BMICat, .fun = length)))+
  geom_bar()

Grouped Bar Chart

  • Grouped bar charts are used to compare multiple categories across multiple groups
data |> 
  ggplot(aes(x = BMICat, fill = Gender))+
  geom_bar()

Distribution Plot: Histogram & Density Plot

Simple Histogram

  • binwidthrefers the width of the interval.
  • If binwidth = 2, and your data ranges from 0 to 10, the bins would be: 0–2, 2–4, 4–6, 6–8, 8–10.
data |> 
  ggplot(aes(x = BMI)) + 
  geom_histogram(color = "black", binwidth = 2)

Grouped Histogram

  • In position it can be identity, stack, dodge, fill
data |> 
  ggplot(aes(x = BMI, fill = Gender)) + 
  geom_histogram(color = "black", binwidth = 2, position = "stack")

Grouped in multiple plot

  • facet_wrap is a function that can divide two groups.
  • In this function default wrapping variable must be included.
data |> 
  ggplot(aes(x = BMI, fill = Gender)) + 
  geom_histogram(color = "black", bins = 10, binwidth = 2)+
  facet_wrap(~Gender)

Simple Density Plot

data |> 
  ggplot(aes(x = BMI)) + 
  geom_density()

Grouped Density plot

data |> 
  ggplot(aes(x = BMI, fill = Gender)) + 
  geom_density()

Grouped in multiple plot

data |> 
  ggplot(aes(x = BMI, fill = Gender)) + 
  geom_density()+
  facet_wrap(~Gender)

Histgram with Density Curve

data |> 
  ggplot(aes(x = BMI, fill = Gender, color = Gender))+
  geom_histogram(aes(y = ..density..), alpha = 0.5, binwidth = 2, bins = 10, position = "identity")+
  geom_density(alpha = .5)+
  facet_wrap(~Gender)

Customization

  • Customization is more or less same in all plot conducted through ggplot2 package.

Simple Customization

data |> 
  ggplot(aes(x = BMICat, fill = Gender))+
  geom_bar()+
  labs( x = "BMI Category",
        y = "Frequency",
        title = "Distribution of BMI category across Gender",
        subtitle = "This figure shows the BMI category based on gender",
        caption = "Data Source: CHIRAL")

Themes (Step: 3)

  • Now we can simply perform theme
# Now we can just add themes with the variable
 Visual + theme_economist()

Available Color Fill

Visual + scale_fill_colorblind()

Manual Color Fill

Visual + scale_fill_manual(values = c("#0868ac", "#006d2c"))

Set font size & family

Visual + theme_base(base_size = 11, base_family = "Algerian")

Font size for labels, tick labels, and legend separately

  • Both
Visual + theme(axis.title = element_text(size = 40))

Font size for labels, tick labels, and legend separately

  • only x-axis
Visual + theme(axis.title.x = element_text(size = 40))

Font size for title and subtitle

# basic + theme(plot.title = element_text(size = 40),  plot.subtitle = element_text(size = 40)

Customize legend

Visual + theme(legend.position = "top")
